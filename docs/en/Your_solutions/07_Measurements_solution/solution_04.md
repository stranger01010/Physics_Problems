# 🔋 Question (Relative Uncertainty)

A car's speedometer has a **5% uncertainty**.  
If it reads **60 km/h**, what is the range of the car's actual speed?

---

# 🧠 What is this problem about?

This question is about **percentage uncertainty (relative uncertainty)**.

We use it to find how much a measurement can vary around its given value.

---

# 📌 Key Idea

A **5% uncertainty** means:

→ The value can increase or decrease by 5% of the measured value.

So we first convert percentage uncertainty into an actual number (km/h).

---

# 🔢 Step 1 — Find the uncertainty amount

We calculate 5% of 60 because we need the actual error in km/h.

Uncertainty = (5 / 100) × 60

Uncertainty = 0.05 × 60

Uncertainty = 3 km/h

👉 Meaning:
The speed can vary by 3 km/h above or below the measured value.

---

# 🔢 Step 2 — Find the lower limit

We subtract the uncertainty because we want the **minimum possible value**.

Lower bound = 60 − 3

Lower bound = 57 km/h

👉 Meaning:
The car could actually be as slow as 57 km/h.

---

# 🔢 Step 3 — Find the upper limit

We add the uncertainty because we want the **maximum possible value**.

Upper bound = 60 + 3

Upper bound = 63 km/h

👉 Meaning:
The car could actually be as fast as 63 km/h.

---

# 📊 Final Answer

Actual speed lies in the range:

57 km/h to 63 km/h

or

57 ≤ speed ≤ 63 km/h

---

# 🎤 Final Explanation (for presentation)

A 5% uncertainty means the measurement may vary by 5% above or below the measured value.  
For 60 km/h, this equals ±3 km/h.  
So the actual speed must be between 57 km/h and 63 km/h.
